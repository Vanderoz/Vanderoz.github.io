Terraria

Terraria is a good game, which can be compared to a more popular game "Minecraft". the game is a little harder and a bit complex compared to the other but overcoming the difficulties is a great way to know you're improving.

The game's bosses are hard to deal with so you need to use your environment for strategies, but you can also invite your friends over for more fire power.

The character you make is a universal one, where you can use the same character for different worlds.
